
Readme for keras_catch
=========================
AHN, Feb 2020

from https://edersantana.github.io/articles/keras_rl

On a 10x10 grid, a ball drops. The paddle at the bottom has to move to catch it.
Train an agent to play this using Q-learning.
Generate a gif with some example games of catch.

$ python qlearn.py
$ python test.py
$ ffmpeg -framerate 1 -i %03d.png output.gif
